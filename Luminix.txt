Objectives:

To design a simple expressive C-like programming language.
To reduce syntax complexity for beginner programmers.
To demonstrate core compiler phases using Flex and Bison.
To translate Luminix code into valid and executable C code.

Language Name: Luminix - bright, learning-focused, futuristic.

Luminix is a beginner-friendly, educational, C-like programming language designed for easy understanding and structured learning. It provides intuitive keywords for data types, control structures, functions, and operators while preserving the logical structure of C. Each Luminix construct has a direct and transparent equivalent in C, making it ideal for learning programming fundamentals and compiler concepts.

Tools:

Flex (Lexical Analyzer):
Used to tokenize source code, identifying keywords, identifiers, constants, operators, and symbols.

Bison (Syntax Analyzer):
Used to define grammar rules, validate syntax, and construct parse trees for programs.

C Language:
Used as the target language for code generation.

Example Code:

Luminix Code

#Attach <stdio.h>

StartExam() {

    Number x;

    Show("Enter a number: ");
    Take("%d", &x);

    Check(x % 2 == 0) {

        Show("Even number");
    }
    Otherwise {

        Show("Odd number");
    }

    Return 0;
}



Equivalent C Code

#include <stdio.h>

int main() {

    int x;

    printf("Enter a number: ");
    scanf("%d", &x);

    if (x % 2 == 0) {

        printf("Even number");

    } else {

        printf("Odd number");
    }

    return 0;
}



Expected Outcomes:

• A fully functional mini-compiler for the Luminix language.
• Error-free translation of Luminix programs into C.
• Improved understanding of compiler design concepts.
• A practical educational tool for beginner programmers.


Conclusion:

Luminix successfully bridges the gap between beginner-friendly programming and professional-level C programming. By combining simplified syntax with a direct C mapping, the language allows learners to focus on problem-solving and logic rather than syntax complexity. The use of Flex and Bison ensures a structured and modular compiler design, making the project both educational and technically sound.
